# Pokédex en Python 

Proyecto 4: Construye una Pokédex

## ¿Que hace?

Primero tenemos que intalar la libreria con el siguiente codigo en la terminal de Python, justo dentro del folder donde estamos guardando
los archivos ".py"

pip install requests

Cuando ejecutas el archivo `ovidio_rodriguez_proyectoM4 te pide el nombre de un Pokemon. Si ese Pokemon existe, la aplicacion:
* Consulta la PokeAPI (https://pokeapi.co)
* Muestra en pantalla el peso, altura, tipos, habilidades, movimientos y la URL de su imagen
* Guarda toda esa informacion en un archivo `.json` dentro de la carpeta donde tengas guardado el archivo que estas ejecutando

Si el nombre esta mal escrito o no existe, muestra un mensaje de error.
